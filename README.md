# TechDex
Trabalho Discente Efetivo para a disciplina de Programação WEB.

Realizado em conjunto com Maria Paula de Jesus e Pedro da Fonseca.

Trata-se de uma plataforma de estudos do meio da Computação. Conta com o formato de catálogo do mundo tech, com uma vasta coleção de materiais. Tendo a oportunidade de criar a sua própria biblioteca, assim fazer um estudo proveitoso e individual. O projeto conta com a relação com o banco de dados relacional MySQL.
